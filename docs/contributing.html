# Contributing Amphib is built on Hiring Agent — and we keep building. Contributions welcome. ## Bugs Open an issue. Include: description, environment (OS, Python version, commit), steps to reproduce, and logs. ## Features Check existing requests first, then open a new one with the problem and your proposed solution. ## Code 1. Open an issue to discuss the change. 2. Fork, branch, code. 3. Format with [Black](https://black.readthedocs.io/en/stable/). 4. Keep prompts provider-agnostic. 5. Test with a real resume on Ollama (and Gemini if you have a key). 6. Open a PR. ## Commit style ``` fix: handle en dash date ranges in work parser feat: add configurable scoring weights ``` Reference the issue number when applicable. ## Conduct Be respectful. Report issues to the maintainers.
