Amphib 🐸

Amphib ingests a resume PDF, extracts structured data with an LLM, cross-references GitHub for ground truth, and produces an explainable score.

📄

PDF Parsing

Converts resume PDFs to clean Markdown while preserving structure — headings, links, tables, all intact.

🧠

LLM Extraction

Section-by-section extraction using Jinja-templated prompts. Works with Ollama (local) or Gemini.

🐙

GitHub Enrichment

Fetches real profile and repo data, classifies projects, and picks the most meaningful contributions.

📊

Fair Scoring

Objective evaluation across four axes with evidence, bonus points, and deductions — no black boxes.

🔒

Offline-First

Run fully local with Ollama. Your data never leaves your machine unless you choose Gemini.

📦

CSV Export

Development mode caches results and appends to a CSV for batch analysis and iteration.

Quick start

git clone <your-repo-url> && cd amphib
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
ollama pull gemma3:4b
cp .env.example .env
python score.py resume.pdf

See the Setup guide for detailed instructions.

Why Amphib?

Because it lives in two worlds — the structured world of resumes and the messy reality of GitHub — and it's comfortable in both.
