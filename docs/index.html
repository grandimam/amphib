Open source · Built on Hiring Agent

Parse resumes.
Extract data. Score candidates.

Three commands turn a PDF resume into structured data and evaluation reports. All from your terminal.

Install
$ curl -sSL https://raw.githubusercontent.com/grandimam/amphib/main/install.sh | bash
amphib analyze Detect layout, columns, section order amphib extract Extract structured data in JSON Resume format amphib evaluate Score across 4 categories with evidence

PDF → Markdown → LLM → Structured Data → Rich TUI

PDFParser converts each page to clean Markdown. The Amphib orchestrator sends it to an LLM with Jinja2 prompts. Results render as tables, trees, panels, and color-coded score bars.
