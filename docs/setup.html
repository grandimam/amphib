# Setup ## Prerequisites - Python 3.11+ - [Ollama](https://ollama.com/) (recommended) or a [Gemini API key](https://aistudio.google.com/api-keys) ## Install ```bash git clone && cd amphib python -m venv .venv && source .venv/bin/activate pip install -r requirements.txt ``` ## Pull a model ```bash ollama pull gemma3:4b # balanced ollama pull gemma3:12b # more capable ollama pull gemma3:1b # lightweight ``` ## Configure ```bash cp .env.example .env ``` Edit `.env`: | Variable | Values | Description | |---|---|---| | `LLM_PROVIDER` | `ollama` or `gemini` | Backend choice | | `DEFAULT_MODEL` | e.g. `gemma3:4b` | Model name | | `GEMINI_API_KEY` | string | Required for Gemini | | `GITHUB_TOKEN` | string | Optional — higher API rate limits | ## Run ```bash python score.py /path/to/resume.pdf ``` That's it. Amphib will parse, enrich, and print a scored evaluation.
